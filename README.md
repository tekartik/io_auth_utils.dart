# io_auth_utils

Io auth helper for google apis.

## Usage

In your `pubspec.yaml`:

```yaml
dependencies:
  tekartik_io_auth_utils:
    git:
      url: git://github.com/tekartik/io_auth_utils.dart
      ref: null_safety
    version: '>=0.3.0'
```

## Helpers

* Load io AuthClient from clientId/secret with console login using links
